1. Запит для побудови архітектури Django-застосунку
Запит до ШІ:
як працює django архітектура і як правильно організувати models views templates urls?
Результат:
ШІ пояснив архітектуру MTV (Model-Template-View), принцип обробки HTTP-запитів та роль кожного компонента. Я використала це для побудови структури свого проєкту та розподілу логіки між файлами.
Фрагмент коду, отриманий від ШІ:
def example_view(request):
    data = Model.objects.all()
    return render(request, 'template.html', {'data': data})
